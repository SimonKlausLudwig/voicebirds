Gatsby Skeleton

Welcome to Gatsby Skeleton

This is a barebones starter intended to be composed by several themes, depending on needs.

It comes with TypeScript, ESLint, Prettier, Husky and Lint Staged configured by default so that the main focus can be set to development right away.

To start working on your site, just create a new file under src/pages, named either index.jsx or index.tsx depending if you wish to use TypeScript or not.

List of available themes:

  • gatsby-theme-skeleton-core — Contains basic information on how to use the skeleton themes.
  • gatsby-theme-skeleton-seo — Includes a Layout component that can be imported to set custom metadata for each of your pages.
